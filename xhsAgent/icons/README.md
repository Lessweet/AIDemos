# 图标说明

请在此文件夹中放置插件图标文件：

- **icon16.png** - 16x16 像素
- **icon48.png** - 48x48 像素
- **icon128.png** - 128x128 像素

## 设计建议

- 使用小红书的品牌色：红色/粉色 (#FF2442)
- 简洁的AI或机器人图标
- 背景透明或白色
- 避免使用蓝色、紫色、渐变色

## 快速生成图标

如果暂时没有图标，插件仍然可以正常运行，只是在浏览器扩展栏中不会显示图标。

你可以使用以下工具快速生成图标：
- [Favicon Generator](https://favicon.io/)
- [Canva](https://www.canva.com/)
- AI生成工具（如 DALL-E, Midjourney）

## 临时解决方案

如果需要快速测试插件，可以：
1. 创建一个简单的纯色PNG图片
2. 或者从网上下载一个AI/机器人图标
3. 使用图片编辑工具调整到对应尺寸
