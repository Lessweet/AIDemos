# UI 设计师面试 AI Agent - 实现计划

## 项目概述

为 UI 设计师求职者打造的智能面试助手，帮助评估职业方向、岗位匹配度，并提供模拟面试功能。

---

## 技术选型（已决定）

| 类别 | 选择 | 理由 |
|------|------|------|
| 前端框架 | Next.js 14 | 全栈框架，部署简单 |
| UI 组件 | Tailwind CSS | 符合黑白简约设计风格 |
| AI 服务 | OpenAI GPT-4 | 成熟稳定，支持文档分析 |
| PDF 解析 | pdf-parse | 轻量级 PDF 文本提取 |
| 图表可视化 | Recharts | React 友好的图表库 |
| 部署平台 | Vercel | 一键部署，免费额度充足 |

---

## 功能模块拆分

### 模块 1：简历与作品集分析
- 上传 PDF 文件（简历 + 作品集）
- AI 解析内容，提取关键信息
- 生成 100 个推荐领域/岗位排行榜
- 可视化表格展示

### 模块 2：岗位 JD 匹配
- 上传或粘贴岗位 JD
- AI 分析匹配度（百分比 + 详细说明）
- 显示优势、劣势、建议

### 模块 3：岗位排行榜
- 实时记录所有评估过的岗位
- 按匹配度排序
- 支持筛选和搜索

### 模块 4：模拟面试
- 基于岗位 JD 生成面试问题
- 文字对话形式进行面试
- 面试结束后生成详细评估报告

## 页面结构

```
/                    → 首页（引导上传简历）
/analysis            → 简历分析结果页

```

## UI 设计规范（来自需求文档）

- 背景色：`#ffffff`
- 主要文字：`#111111`
- 边框：`#000000`（2px solid）

- 过渡动画：`all 0.3s ease`
