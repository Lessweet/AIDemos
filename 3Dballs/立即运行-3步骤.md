# 🚀 立即运行 - 只需3步（2分钟）

旧项目文件已损坏已删除。现在按以下步骤创建新项目：

---

## 第1步：在Xcode创建新项目（30秒）

1. **打开Xcode**

2. **点击 "Create a new Xcode project"**

3. **选择模板：**
   - 平台：iOS
   - 应用类型：**App**
   - 点击 Next

4. **填写项目信息：**
   ```
   Product Name: GlassBalls          ⬅️ 重要！用这个名字
   Team: (选择你的开发者账号)
   Organization Identifier: com.yourname
   Interface: SwiftUI                ⬅️ 必须选SwiftUI
   Language: Swift                   ⬅️ 必须选Swift
   ```
   取消勾选 Include Tests（不需要测试）

5. **保存位置：**
   ```
   /Users/chentongrong/Documents/workspace/AIDemos/3Dballs/
   ```
   点击 Create

---

## 第2步：替换代码文件（30秒）

在Xcode左侧项目导航器中：

### 删除默认文件：
1. 右键点击 **`ContentView.swift`** → Delete → Move to Trash
2. 右键点击 **`GlassBallsApp.swift`**（或其他App.swift）→ Delete → Move to Trash

### 添加新文件：
3. 右键点击 **GlassBalls 文件夹**（黄色图标）
4. 选择 **"Add Files to 'GlassBalls'..."**
5. 导航到：`/Users/chentongrong/Documents/workspace/AIDemos/3Dballs/`
6. **按住Command键**，同时选择这3个文件：
   - ✅ GlassBallsApp.swift
   - ✅ GlassBallsView.swift
   - ✅ GlassBallsScene.swift
7. **勾选 "Copy items if needed"**
8. 点击 **Add**

---

## 第3步：添加权限并运行（1分钟）

### 添加陀螺仪权限：
1. 点击左侧最顶部的**蓝色项目图标**（GlassBalls）
2. 在中间选择 **TARGETS → GlassBalls**
3. 点击 **Info** 标签页
4. 找到任意一行，点击 **+** 号
5. 在弹出的下拉框中输入并选择：**`Privacy - Motion Usage Description`**
6. 右侧值填写：`需要访问陀螺仪来实现玻璃球重力效果`

### 连接iPhone运行：
7. 用数据线连接你的iPhone
8. 在Xcode顶部中间位置，点击设备选择器（可能显示"Any iOS Device"）
9. 选择你的iPhone设备
10. 点击左上角的 **播放按钮 ▶️**（或按 Command + R）

### 首次运行：
- iPhone上可能提示"不受信任的开发者"
- 去 iPhone 设置 → 通用 → VPN与设备管理 → 点击你的Apple ID → 信任

## 🐛 遇到问题？

### Build失败 - "No such module 'SpriteKit'"
- 重启Xcode

### 文件后面有问号
- 不影响，这是Git标记，忽略即可

### 显示"Hello, world!"
- 确保删除了所有旧的App入口文件
- 确保3个Swift文件都添加到项目了
- 按 Command + Shift + K 清理，再重新运行

### 白屏或黑屏但没有球
- 检查是否在真机上运行（模拟器无陀螺仪）
- 检查Target Membership（选中文件，右侧Inspector要勾选项目名）
