# 作为 UI 设计师，我花了 2 天 Vibe coding 了小红书 创作者 回复评论 Agent 

## 为什么要开发AI 回复插件？

最近在小红书上持续分享AI 设计相关内容，获得了意料之外的增长，在赞、藏、评论，我感觉我的内容发挥了最大价值；

同时，作为内容创作者，我也遇到了一些阻力，主要在回复评论上我有压力：

1、作为i人的我，自带小小的社交压力，每次回复之前，都要花费时间咬文嚼字，组织语言、调整语气、抠措辞，希望以「完美的人设」回复网友
2、有时遇到网友的评论内容涉及到我的知识盲区，我就不知道如何回答
3、通常做法：把原文内容和网友的回复内容同时提供给 ChatGPT，让它帮我生成几条回复，我参考之后再编辑给网友回复，这样的话，其实我相当于在小红书平台和 ChatGPT 之间来回跳转。虽然我日常是电脑工作，来回切换页面也挺麻烦的 

## 设计目标：省时、省力、省步骤

因此，我就想能不能直接用AI工具做个插件，帮我快速生成内容。 正好，我有个 Cloud Code ，于是我尝试把我的背景、问题、目标、设计想法写成prompt，丢给Cloud Code，反复调试了几版后，获得了一版UI最符合我预期、且交互效果超预期的prompt
