# 快速开始指南

## 5分钟在iPhone上运行这个应用

### 步骤1：打开Xcode

1. 在Mac上打开Xcode
2. 点击 "Create a new Xcode project"

### 步骤2：创建项目

1. 选择 **iOS** → **App**
2. 点击 **Next**
3. 填写信息：
   ```
   Product Name: Card3D
   Team: （选择你的团队或个人账号）
   Organization Identifier: com.yourname
   Interface: SwiftUI ✓
   Language: Swift ✓
   ```
4. 点击 **Next**，选择保存位置

### 步骤3：替换代码文件

1. **删除自动生成的文件**：
   - 在项目导航器中，删除 `ContentView.swift`
   - 删除自动生成的App文件（名称类似 `Card3DApp.swift`）

2. **添加我们的文件**：
   - 将这4个Swift文件拖入Xcode项目：
     - ✅ `Card3DApp.swift`
     - ✅ `ContentView.swift`
     - ✅ `Card3DView.swift`
     - ✅ `ControlPanelView.swift`

   - 拖入时选择：
     - ✅ Copy items if needed
     - ✅ Add to targets: Card3D

### 步骤4：运行应用

#### 在模拟器上运行：
1. Xcode顶部选择模拟器（推荐：iPhone 14 Pro）
2. 点击 ▶️ 按钮（或按 `Cmd + R`）
3. 等待编译和启动
4. 开始体验！

#### 在真机上运行（推荐）：
1. 用数据线连接iPhone到Mac
2. iPhone上点击"信任此电脑"
3. Xcode顶部选择你的iPhone设备
4. 点击 ▶️ 运行
5. 首次运行需要信任证书：
   - iPhone：设置 → 通用 → VPN与设备管理
   - 找到你的开发者账号，点击"信任"
6. 重新运行应用

## 常见问题解决

### 问题1：没有开发团队
**解决方法**：
1. Xcode → Preferences → Accounts
2. 点击 + 号添加Apple ID
3. 使用免费的个人开发者账号即可

### 问题2：编译错误
**检查清单**：
- ✅ 确保4个文件都已添加到项目
- ✅ 确保Interface选择了SwiftUI
- ✅ 确保Language选择了Swift
- ✅ iOS部署目标设置为15.0或以上

### 问题3：真机无法运行
**解决方法**：
1. 检查数据线连接
2. 检查iPhone是否信任此电脑
3. 检查开发者证书是否信任
4. 尝试重启Xcode和iPhone

## 使用技巧

### 基本操作
- 👆 **拖动卡片**：单指拖动旋转
- 📱 **打开面板**：点击底部"参数控制"
- 🎚️ **调整参数**：滑动滑块实时预览
- 💾 **保存设置**：点击"保存"按钮

### 最佳体验
- 竖屏使用效果最佳
- 建议在真机上运行（体验更流畅）
- 调整参数后可以看到即时效果
- 拖动卡片时自动旋转会暂停
