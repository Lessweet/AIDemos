# 3D交互式卡片 - iOS原生应用

基于SwiftUI开发的3D交互式卡片iOS原生应用，提供流畅的原生体验。

## 功能特性

### 核心功能
- ✅ 3D卡片在iPhone框架内展示
- ✅ 流畅的拖动旋转交互
- ✅ 自动旋转动画
- ✅ 完整的参数控制面板
- ✅ 参数本地保存
- ✅ 原生iOS体验

### 交互体验
- **拖动旋转**：手指拖动卡片进行360度旋转
- **自动旋转**：卡片自动3D旋转展示
- **实时调整**：滑动调整参数即时预览效果
- **参数保存**：保存设置下次自动加载

## 如何在Xcode中运行

### 方法1：创建新的Xcode项目（推荐）

1. **打开Xcode**
   - 启动Xcode
   - 选择 "Create a new Xcode project"

2. **选择模板**
   - iOS → App
   - 点击 "Next"

3. **配置项目**
   - Product Name: `Card3D`
   - Team: 选择你的开发团队
   - Organization Identifier: `com.yourname`
   - Interface: **SwiftUI**
   - Language: **Swift**
   - 点击 "Next" 并选择保存位置

4. **替换文件**
   - 删除自动生成的 `ContentView.swift`
   - 将以下4个Swift文件拖入项目：
     - `Card3DApp.swift` → 替换自动生成的App文件
     - `ContentView.swift`
     - `Card3DView.swift`
     - `ControlPanelView.swift`

5. **运行应用**
   - 选择模拟器（iPhone 14 Pro 推荐）
   - 点击 ▶️ 运行按钮
   - 或按快捷键 `Cmd + R`

### 方法2：在真机上运行

1. **连接iPhone**
   - 用数据线连接iPhone到Mac
   - iPhone上信任此电脑

2. **选择设备**
   - Xcode顶部选择你的iPhone设备

3. **运行**
   - 点击 ▶️ 运行
   - 首次运行需要在iPhone上信任开发者证书：
     - 设置 → 通用 → VPN与设备管理 → 信任

## 项目结构

```
Card3D/
├── Card3DApp.swift          # 应用入口
├── ContentView.swift        # 主视图（iPhone框架+卡片）
├── Card3DView.swift         # 3D卡片视图和数据模型
└── ControlPanelView.swift   # 参数控制面板
```

## 文件说明

### Card3DApp.swift
- 应用程序入口
- 配置应用生命周期

### ContentView.swift
- 主界面布局
- iPhone设备框架展示
- 控制面板弹出动画
- 整合卡片和控制面板

### Card3DView.swift
- 3D卡片视图组件
- 卡片数据模型（CardModel）
- 拖动旋转手势处理
- 自动旋转动画逻辑
- 参数持久化存储

### ControlPanelView.swift
- 参数控制面板UI
- 样式参数滑块
- 动画参数滑块
- 交互参数滑块
- 控制按钮（暂停、重置、保存）

## 参数说明

### 样式参数
- **卡片宽度**：250-370px
- **卡片高度**：150-320px
- **圆角大小**：0-50px
- **阴影大小**：10-80px

### 动画参数
- **自动旋转速度**：0-2倍速
- **左右摇摆幅度**：0-50度
- **上下倾斜幅度**：0-50度
- **动画流畅度**：0.01-0.5

### 交互参数
- **拖动灵敏度**：0.1-2
- **最大旋转角度**：15-90度

## 使用指南

### 基本操作
1. **拖动卡片**：手指按住卡片拖动旋转
2. **查看参数**：点击底部"参数控制"按钮
3. **调整参数**：滑动各个参数滑块
4. **保存设置**：点击"保存"按钮

### 控制按钮
- **暂停/继续**：控制自动旋转
- **重置**：恢复所有参数到默认值
- **保存**：保存当前参数设置

### 提示
- 调整参数后会立即生效
- 保存的参数会在下次启动时自动加载
- 拖动卡片时会暂停自动旋转
- 可以随时点击"重置"恢复初始状态

**Q: 如何分发给其他人？**
