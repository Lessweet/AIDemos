# 快速开始 - 3 步运行项目

## 第 1 步：打开 Xcode 创建项目

1. 打开 **Xcode**
2. 选择 **File** > **New** > **Project**
3. 选择 **iOS** > **App**，点击 **Next**
4. 配置：
   - Product Name: `Sphere3D`
   - Interface: **SwiftUI**
   - Language: **Swift**
5. 保存到：`/Users/chentongrong/Documents/workspace/AIDemos/3Dsphere`
6. 点击 **Create**

## 第 2 步：添加源代码文件

1. 在 Xcode 左侧项目导航器中，**删除** `ContentView.swift`
2. 将以下 3 个文件拖入 Xcode 项目：
   - `Sphere3D/SphereScene.swift`
   - `Sphere3D/SphereView.swift`
   - `Sphere3D/Sphere3DApp.swift`（替换自动生成的）
3. 弹出对话框时，确保勾选 **"Copy items if needed"**

## 第 3 步：运行

1. 选择 iPhone 模拟器（如 iPhone 15 Pro）
2. 点击 ▶️ 运行按钮
3. 等待编译完成，应用会自动启动

## 效果预览

- 屏幕中央会显示一个**蓝色半透明 3D 球体**
- 球体会**自动旋转**
- 可以**单指拖动**旋转球体
- 可以**双指捏合**缩放球体
- 点击**右上角按钮**切换为不规则边缘球体

## 如果遇到问题

### 编译错误
- 确保选择的是 **iOS** 平台（不是 macOS）
- 确保 Deployment Target 是 **iOS 15.0** 或以上

### 球体不显示
- 检查是否正确添加了所有 3 个 Swift 文件
- 尝试清理项目：**Product** > **Clean Build Folder**

### 手势不响应
- 确保在 **真机或模拟器** 上测试（不是 Mac Catalyst）
