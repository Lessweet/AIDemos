# 使用真实iPhone图片的说明

## 步骤：

1. 将你的iPhone图片保存到项目目录，命名为：`iphone-frame.png`

2. 打开 `3d-card.html` 文件

3. 找到这行代码（大约在第29行）：
```css
background-image: url('data:image/svg+xml;utf8,...');
```

4. 替换为：
```css
background-image: url('iphone-frame.png');
```

5. 根据你的iPhone图片实际尺寸，调整以下参数：
   - `.iphone-frame` 的 `width` 和 `height`
   - `.iphone-screen` 的 `top`、`left`、`width`、`height`

## 屏幕区域坐标说明：

如果你的iPhone图片是标准尺寸（例如 430x878），屏幕区域大约在：
- 左边距：18px
- 上边距：18px
- 宽度：394px
- 高度：842px

你可以根据实际图片微调这些数值，让卡片正好显示在手机屏幕区域内。
